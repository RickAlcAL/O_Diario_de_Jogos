# O_Diario_de_Jogos
Sistema em Java estruturado em POO para persistência de dados textuais. Permite ao usuário definir dinamicamente o caminho do diretório e gerenciar um catálogo de jogos personalizado. O projeto utiliza manipulação de fluxos de I/O (BufferedWriter e Scanner) com tratamento de exceções robusto e padronização de mensagens via Enums.
